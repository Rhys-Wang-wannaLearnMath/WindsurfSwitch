# WindsurfSwitch 使用说明

## 📦 安装

### 方式一：直接安装 VSIX（推荐）

1. 下载 `windsurf-SWITCH-1.0.0.vsix`
2. 在 Windsurf 中按 `Ctrl+Shift+P`
3. 输入 `Extensions: Install from VSIX...`
4. 选择下载的 `.vsix` 文件

### 方式二：从源码构建

```bash
git clone https://github.com/Rhys-Wang-wannaLearnMath/WindsurfSwitch.git
cd WindsurfSwitch
npm install
npm run build
npm run package
```

## 🔐 权限设置（重要）

扩展需要修改 Windsurf 核心文件来实现账号切换功能。**首次使用前必须设置权限**。

### Linux 系统

```bash
# 修改扩展文件权限
sudo chmod 666 /usr/share/windsurf/resources/app/extensions/windsurf/dist/extension.js

# 验证权限（应显示 -rw-rw-rw-）
ls -la /usr/share/windsurf/resources/app/extensions/windsurf/dist/extension.js
```

### macOS 系统

```bash
# 修改扩展文件权限
sudo chmod 666 /Applications/Windsurf.app/Contents/Resources/app/extensions/windsurf/dist/extension.js

# 验证权限
ls -la /Applications/Windsurf.app/Contents/Resources/app/extensions/windsurf/dist/extension.js
```

### Windows 系统

以**管理员身份**运行 PowerShell：

```powershell
# 获取文件路径
$path = "$env:LOCALAPPDATA\Programs\Windsurf\resources\app\extensions\windsurf\dist\extension.js"

# 修改权限
icacls $path /grant Users:F
```

## 🚀 使用方法

### 1. 打开扩展面板
点击左侧 Activity Bar 的 **Windsurf 图标**

### 2. 添加账号
- 点击「**添加账号**」按钮
- 输入 Windsurf 账号的**邮箱**和**密码**
- 扩展会自动获取 API Key 并保存

### 3. 切换账号
- 在账号列表中点击想要切换的账号
- 首次切换会自动应用补丁并提示重启 Windsurf
- 重启后即可使用新账号

### 4. 快捷键切换
| 系统 | 快捷键 | 功能 |
|------|--------|------|
| Windows/Linux | `Ctrl+Alt+S` | 切换到下一个账号 |
| macOS | `Cmd+Alt+S` | 切换到下一个账号 |

### 5. 账号备注
- 在账号列表中，点击账号右侧的 **✏️** 按钮
- 输入备注名（如「工作号」「测试号」），留空可清除备注
- 备注会显示在账号列表和「当前账号」卡片中

### 6. 复制账号密码
- 在账号列表中，点击账号右侧的 **🔐** 按钮
- 会自动复制 `邮箱,密码` 格式到剪贴板
- 注意：仅当添加账号时保存了密码才能使用此功能

### 7. 删除账号
- 点击账号右侧的 **🗑️** 按钮

## ⚠️ 常见问题

### 问题 1：切换失败 - 权限不足

**错误信息**：
```
Insufficient permissions to modify Windsurf extension
```

**解决方案**：
按照上方「权限设置」章节修改文件权限。

---

### 问题 2：切换失败 - 版本不兼容

**错误信息**：
```
Could not find handleAuthToken function. Windsurf version may be incompatible.
```

**解决方案**：
本 Fork 版本已适配 **Windsurf 1.106.0**。如果您使用其他版本，可能需要手动修改 `windsurfPatchService.ts` 中的函数签名。

检查 Windsurf 版本：
```bash
windsurf --version
```

---

### 问题 3：Windsurf 更新后需要重新应用补丁

Windsurf 更新会覆盖 `extension.js` 文件，需要：
1. 重新设置文件权限
2. 重新执行一次账号切换操作

## 📁 补丁文件位置

| 系统 | 路径 |
|------|------|
